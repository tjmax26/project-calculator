# project-calculator
Calculator Project
A simple browser-based calculator built as part of the Foundations course.
About the Project
This calculator was built to practice the core concepts learned in the Foundations course. It performs basic math operations — addition, subtraction, multiplication, and division — right in the browser.
What I Learned
Building this project helped me practice:

Strings — reading button text and updating the display
Arrays — working with lists of buttons
If/else statements — checking which button was clicked and deciding what to do
Functions — writing reusable blocks of code for each math operation
Variables — keeping track of numbers and operators between clicks
DOM manipulation — making buttons interactive with JavaScript

How to Use

Click any number button to enter a number
Click an operator button (+, -, ×, ÷)
Enter a second number
Press = to see the result
Press CLR to reset and start over
